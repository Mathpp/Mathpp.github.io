Experimental mathematical Calculator

Features

  • Backed programmed in C++ and UI in javascript running both offline on your Device
  • Temporary Only German Number Format accepts numbers like 10.000.003,23 and ";" to sepeate parameters in internal Functions
  • define your own operators like a%b:=a mod b
  • Shows steps (To be improved)
  • Switch Numeric Base (b) with "switchbase(b)"
  • Print Result of Expression (e) with Numeric Base (b) with "tobase(e;b)"
